# Тестовое задание для LogicLike

В данном задании был реализован kidstech проект, где дети 5-12 лет учаться думать
В качестве основы был использован дизайн, выполненный в figma

# Стек
Проект выполнен с помощью:
- React
- SASS
- Axios
